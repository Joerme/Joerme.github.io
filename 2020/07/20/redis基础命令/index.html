redis基础命令

数据库 nosql
发布日期:   2020-07-20

环境

初学redis，对于redis的基础命令进行一下记录

个人是在阿里云服务器上用docker配置环境，docker相对比较方便.. 配置好后一行命令进入redis

docker exec -it myredis redis-cli

命令

redis支持五种数据类型：string(字符串),list(列表),hash(哈希),set(集合),sorted set(有序集合)

key-value格式

keys [pattern]根据正则表达式获取键名 keys * 获取所有key

type 键名 获取键值类型

del 键名 删除键值

string

string 类型是二进制安全的。意思是 redis 的 string 可以包含任何数据。比如jpg图片或者序列化的对象。

127.0.0.1:6379> set 4 5
OK
127.0.0.1:6379> get 4
"5"
127.0.0.1:6379> keys *
1)"4"

hash

适合存储对象

hget获取对象某个属性值，hgetall获取对象所有信息

127.0.0.1:6379> hmset hash name kevin age 11
OK
127.0.0.1:6379> hget hash name
"kevin"
127.0.0.1:6379> hgetall hash
1) "name"
2) "kevin"
3) "age"
4) "11"

list

按照插入顺序排序。你可以添加一个元素到列表的左边或者右边

redis 127.0.0.1:6379> LPUSH demolist redis
(integer) 1
redis 127.0.0.1:6379> LPUSH demolist mongodb
(integer) 2
redis 127.0.0.1:6379> LPUSH demolist mysql
(integer) 3
127.0.0.1:6379> lrange demolist 0 10
1) "mysql"
2) "mongodb"
3) "redis"

set

Redis 的 Set 是 String 类型的无序集合。集合成员是唯一的，这就意味着集合中不能出现重复的数据。

redis 127.0.0.1:6379> SADD set redis
(integer) 1
redis 127.0.0.1:6379> SADD set mongodb
(integer) 1
redis 127.0.0.1:6379> SADD set mysql
(integer) 1
redis 127.0.0.1:6379> SADD set mysql
(integer) 0
redis 127.0.0.1:6379> SMEMBERS set

1) "mysql"
2) "mongodb"
3) "redis"

sorted set

redis有序集合和集合一样也是string类型元素的集合,且不允许重复的成员。

不同的是每个元素都会关联一个double类型的分数。redis正是通过分数来为集合中的成员进行从小到大的排序。

redis 127.0.0.1:6379> ZADD runoobkey 1 redis
(integer) 1
redis 127.0.0.1:6379> ZADD runoobkey 2 mongodb
(integer) 1
redis 127.0.0.1:6379> ZRANGE runoobkey 0 10 WITHSCORES

1) "redis"
2) "1"
3) "mongodb"
4) "2"
